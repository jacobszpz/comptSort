# comptSort

A sorting function.

```python
>>> from comptSort import comptSort
>>> from comptSort import SortingAlgorithm as Sorting
>>> uData = [9,8,7,6,5,4,3,2,1]
>>> sData = comptSort(uData, Sorting.MERGE, asc=True)
>>> sData
[1,2,3,4,5,6,7,8,9]
```
